# DkZ Shared Scripts

> **Modular Script Library for 146 Modules** - Part of the [DEVKiTZ](https://github.com/D-VKITZ) Ecosystem

---

## Architecture

```
Module.html
  +-- <script src='dkz-premium.js'>   Premium UI
  +-- <script src='dkz-copilot.js'>   AI Chat+Builder
  +-- <script src='dkz-james.js'>     Guardian QA
  +-- <script src='dkz-navbar.js'>    Navigation
  +-- <script src='dkz-debug.js'>     Dev Tools
  +-- <script src='dkz-guide.js'>     Onboarding
  +-- <script src='dkz-console.js'>   Console

All 146 modules include these 7 scripts.
```

## Scripts

| Script | Size | Features |
|:-------|:-----|:---------|
| dkz-premium.js | Core | LiveTicker, Particle BG, Animations |
| dkz-copilot.js | Core | AI Chat, 18 Providers, Builder |
| dkz-james.js | Core | Guardian QA, Error Tracking |
| dkz-navbar.js | Core | Sidebar, Notes, Search |
| dkz-debug.js | Dev | Console, Inspector, Stats |
| dkz-guide.js | UX | Tooltips, Onboarding Flow |
| dkz-console.js | Dev | Custom Console Commands |
