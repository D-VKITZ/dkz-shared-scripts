# DkZ Shared Scripts

DkZ Shared Scripts — Premium, Debug, Guide, Common Utilities.

## Features

- dkz-theme.css: Global design system with CSS variables
- dkz-navbar.js: Navigation bar with module linking
- dkz-console.js: In-browser developer console
- dkz-debug.js: Debug panel with localStorage viewer
- dkz-guide.js: Onboarding tour system
- dkz-james.js: James Guardian agent in browser

## Tech Stack

JavaScript ES6+ · CSS3 Custom Properties

## Quick Start

```bash
git clone https://github.com/D-VKITZ/dkz-shared-scripts.git
cd dkz-shared-scripts
# Include scripts in your HTML:
# <script src="shared/dkz-navbar.js"></script>
```
